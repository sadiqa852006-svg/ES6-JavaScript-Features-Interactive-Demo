# ES6 Features Interactive Demo

A responsive frontend website that demonstrates modern JavaScript ES6 features using interactive examples.

This project helps beginners understand how ES6 features work in real-time through simple demonstrations.

## 🚀 Features

- Arrow Functions
- Promises
- Async / Await
- Destructuring
- Template Literals
- JavaScript Modules
- Responsive layout using Flexbox
- Interactive example buttons
- Modern dark UI design

## 🛠 Technologies Used

- HTML5
- CSS3 (Flexbox)
- JavaScript ES6
